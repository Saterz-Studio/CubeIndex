# CubeIndex
 
 **CubeIndex** is a website where you can manage your speed (or non speed) cube collection, it is built with **SvelteKit**, **TailwindCSS**, **Supabase**, and more...
 
## Tech Stack
 
 - **Frontend:** SvelteKit and TailwindCSS
 - **Backend:** Supabase
 - **Hosting:** Vercel
